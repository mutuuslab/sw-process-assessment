# 손에 잡히는 SW 개발 프로세스 · 이해도 평가 (웹앱)

> 현대자동차 구매본부 3일 과정 · 협력사 SW 품질 점검·관리 역량 점검용
> 원본 문항: `이해도_평가.docx` (10문항 / 문항당 10점 / 100점 / 합격 70점)

## 📂 파일
- **`이해도평가_웹앱.html`** — 이 파일 하나가 전부입니다. 인터넷 없이 동작합니다.

## ▶ 실행 방법
1. `이해도평가_웹앱.html` 파일을 **더블클릭** (Chrome / Edge 권장)
2. 별도 설치·서버 불필요. USB로 복사해 교육생 PC에 배포해도 됩니다.

---

## 👨‍🎓 교육생 (응시자)
1. 이름(또는 사번) 입력 → **평가 시작**
2. 한 문항씩 풀면 즉시 **AI 학습 코치**가 피드백:
   - 내가 고른 오답이 왜 틀렸는지
   - 정답의 근거 + 핵심 해설
   - **💼 구매 실무 포인트** (협력사 점검 관점)
3. 마지막에 **개인별 진단 리포트** (점수 / 영역별 이해도 막대 / 맞춤 학습 가이드)
4. **⬇ 내 결과 파일로 저장(JSON)** → 강사에게 제출 (다른 PC에서 응시한 경우)

## 👨‍🏫 강사 (대시보드)
홈 화면 → **📊 강사용 대시보드 열기**
- 평균 점수 / 합격률 / 평균 소요시간
- **점수 분포** 그래프
- **문항별 정답률** (낮은 문항 = 다음 차수 보충 설명 대상)
- **영역별 평균 이해도** (가장 약한 영역 자동 안내)
- **응시자별 표** (행 클릭 → 개인 상세 + 진단 리포트)

### 결과 모으기 (2가지 방식)
- **같은 PC(키오스크):** 교육생이 차례로 응시하면 자동 저장(localStorage)
- **여러 PC:** 교육생이 저장한 JSON들을 **⬆ 결과파일 가져오기**로 한 번에 취합 (여러 개 동시 선택 가능, 중복 자동 제외)

## 🤖 (선택) 실시간 AI 튜터
- 기본 해설·진단은 **API 키 없이** 모두 작동합니다.
- 우측 상단 **⚙ 설정**에서 Anthropic API 키 입력 시, 문항 피드백 화면에 **'AI 튜터에게 추가 질문'** 채팅이 켜집니다.
- 키는 해당 브라우저에만 저장되며, 질문 시 Anthropic API로만 직접 호출됩니다.
- ⚠ 사내 보안정책상 외부 API가 차단되면 내장형 해설만 사용하세요(기능에 지장 없음).

## 🔧 문항 수정
HTML 파일 상단 `const QUESTIONS = [...]` 배열을 편집하면 문항·정답·해설·실무 포인트를 바꿀 수 있습니다.
- `answer`: 정답 인덱스(0=①, 1=②, 2=③, 3=④)
- `why`: 보기 4개 각각의 진단(정답/오답 사유)
- `core`: 핵심 해설 / `realworld`: 구매 실무 포인트
